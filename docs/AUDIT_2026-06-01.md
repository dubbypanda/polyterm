# PolyTerm Audit - June 1, 2026

## Scope

This audit checked PolyTerm against current public Polymarket API behavior and the repository's own command, docs, and test gates. The focus was runtime correctness for market discovery, CLOB token routing, public trade history, live websocket monitoring, CLI/TUI command registration, and workspace validation.

## Current Polymarket Standards Applied

- Gamma market discovery uses `https://gamma-api.polymarket.com/markets/keyset`; legacy `/markets` offset pagination is deprecated and returns `Deprecation: true` with a May 1, 2026 sunset header.
- Gamma slug lookup uses `/markets/slug/{slug}`. Numeric IDs still use `/markets/{id}`.
- Gamma search uses `/public-search` with `q` and `limit_per_type`.
- Removed Gamma nested routes such as `/markets/{id}/prices`, `/markets/{id}/volume`, and `/markets/{id}/trades` are not used.
- CLOB order book, price, spread, fee-rate, and history endpoints receive CLOB token IDs, not Gamma market IDs.
- Public historical trade polling uses Data API `/trades`.
- Live trade/orderbook streaming uses `wss://ws-subscriptions-clob.polymarket.com/ws/market`; websocket payloads may arrive as JSON batches.

## Fixes Made

- Added `polyterm.api.market_utils` for token ID, condition ID, slug, list-field, and metadata-price normalization.
- Updated Gamma client search, slug lookup, metadata-derived price/volume helpers, and Data API-backed trade history.
- Updated scanner and aggregator CLOB calls to resolve token IDs before hitting token-only endpoints.
- Updated the default configured trade websocket endpoint to the CLOB market websocket.
- Updated CLOB websocket listeners to parse both single-message and batched-message payloads.
- Moved whale tracker REST fallback to Data API trade polling and added Data API wallet-shape support.
- Added `-i` as the documented `live-monitor --interactive` alias.
- Corrected docs for live monitor websocket behavior, TUI shortcuts, chart `--sparkline`, config defaults, and API endpoint contracts.
- Replaced `test_all_commands.sh` with a checkout-bound, macOS-portable command smoke script.
- Added documentation alias files so `scripts/validate_docs.py` has no blocking missing-doc or broken-link errors.

## Live Verification

Live endpoint probes passed on June 1, 2026:

- `GammaClient.get_markets(limit=1, active=True, closed=False)` returned active market data from `/markets/keyset`.
- `GammaClient.get_market(slug)` resolved the same market through `/markets/slug/{slug}`.
- `GammaClient.get_market_prices(id)` derived a current price from market metadata.
- `GammaClient.get_market_volume(id)` returned market metadata volume fields.
- `GammaClient.get_market_trades(id, limit=2)` returned two public rows through Data API `/trades`.
- `GammaClient.search_markets("bitcoin", limit=2)` returned two rows from `/public-search`.
- `CLOBClient.get_order_book(token_id, depth=2)` returned bids and asks.
- CLOB market websocket subscription returned a batched `book` payload from `wss://ws-subscriptions-clob.polymarket.com/ws/market`.
